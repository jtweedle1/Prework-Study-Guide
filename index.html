Prework Study Guide

HTML

  • Hypertext markup language (HTML): one of the three cornerstone languages of web development.
  • Metadata: information about our webpage that the browser uses to effectively serve it to users.
  • Search engine optimization (SEO): the process of improving the ranking of a website in search engines.

CSS

  • CSS stands for "Cascading Style Sheets" and is often used to add style to a webpage after setting up the structure with HTML.

Git

  • GitHub: web-based interface that uses Git which allows multiple people to make separate changes to a program at the same time.
  • Every command using Git must be preceded by "git" in the command line interface.
  • git status: Checks what branch we are currently on.
  • git checkout -b branch-name: Creates a new branch and switches to it.

JavaScript

  • JavaScript adds logic to webpages, allowing users to interact with it.
